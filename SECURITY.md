# Security Policy

## Reporting a Vulnerability

If you discover a security vulnerability in this project, please report it responsibly.

**Do NOT open a public GitHub issue for security vulnerabilities.**

Instead, please report vulnerabilities to Snowflake's security team:

- **Email:** [security@snowflake.com](mailto:security@snowflake.com)

Include the following in your report:

- Description of the vulnerability
- Steps to reproduce
- Potential impact
- Any suggested fixes (optional)

The Snowflake security team will acknowledge your report and work with you to address the issue.

## Scope

This repository is a **sample/demo project** intended to demonstrate Snowflake-native ontology
concepts. It is not a supported Snowflake product. However, we still take security seriously
and appreciate responsible disclosure of any issues.
